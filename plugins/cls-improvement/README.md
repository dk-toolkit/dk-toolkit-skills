# CLS Improvement

> Detect, analyze, plan, implement, and validate Cumulative Layout Shift improvements.

## What It Does

CLS Improvement follows a 12-step workflow to reduce layout shift on your Shopify store. It collects target URLs, generates before-reports via PageSpeed Insights, analyzes the root causes, builds an improvement plan, executes the fixes, re-tests with after-reports, compares results, and logs learnings for future runs.

## Triggers

- `/cls-improvement`

## Requirements

- Google PageSpeed Insights API key

## Quick Start

Run `/cls-improvement` and provide the pages you want to optimize. The plugin will measure current CLS scores, identify shift sources, apply fixes, and verify the improvements automatically.
