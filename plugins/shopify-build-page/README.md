# Shopify Build Page

> Builds complete Shopify theme pages from Figma designs (desktop + mobile).

## What It Does

Shopify Build Page takes a Figma design and turns it into a fully functional Shopify theme page. It splits the design into logical sections, generates each one via the section-generator plugin, wires everything into the template JSON, and runs QA to verify the final output matches the original design.

## Triggers

- "build page from figma"
- "create shopify page"

## Requirements

- Figma Desktop MCP
- Shopify Dev MCP (optional)

## Quick Start

Open a Figma file containing both desktop and mobile frames, then use one of the trigger phrases. The plugin will walk through section splitting, code generation, template wiring, and QA automatically.
