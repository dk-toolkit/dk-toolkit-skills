# Shopify Section Generator

> Generates a single Shopify theme section (Liquid + schema + CSS + JS) from Figma or CSS dumps.

## What It Does

Shopify Section Generator produces a complete, production-ready Shopify section from a Figma frame or a raw CSS dump. It follows Dawn / OS 2.0 conventions, writes mobile-first CSS, and wraps JavaScript in the Web Component pattern. The output includes the Liquid template, JSON schema, scoped styles, and any required JS.

## Triggers

- "generate shopify section"
- "create section from figma"

## Requirements

- Figma Desktop MCP
- Shopify Dev MCP (optional)

## Quick Start

Select or reference a Figma frame (or paste a CSS dump), then use a trigger phrase. The plugin will generate the full section file ready to drop into your theme's `sections/` directory.
