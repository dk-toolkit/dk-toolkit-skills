# Visual QA

> Automated Shopify theme visual QA across 11 breakpoints (320px-2560px).

## What It Does

Visual QA tests your Shopify theme at 11 viewport widths from 320px to 2560px. It checks for horizontal overflow, incorrect container widths, broken gutter resets, CSS scope leaks, and image rendering issues, then reports every finding with the exact breakpoint and element involved.

## Triggers

- "visual qa"
- "test breakpoints"
- "check responsive"

## Requirements

- No external dependencies (uses built-in browser automation)

## Quick Start

Use one of the trigger phrases and provide the page URL. The plugin will cycle through all 11 breakpoints, run its checks, and return a report of any issues found.
